# Serveye: Restaurant Analytics Dashboard

An intelligent performance tool that transforms raw restaurant data into actionable growth strategies using AI.

## Key Features
* **AI-Generated Recommendations:** Integrated with the **OpenAI API** to provide data-driven business advice and trend analysis.
* **Performance Analytics:** Visual tracking of peak hours, staff efficiency, and menu item performance.
* **Insightful Dashboard:** A centralized hub for viewing complex datasets in an easy-to-digest format.

## Tech Stack
* **Language:** Java
* **AI Integration:** OpenAI API
* **Data Processing:** JSON / SQL
